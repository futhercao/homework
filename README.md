# 信息与知识获取 — 课程作业交付

本仓库交付**两个相互独立、各自可运行**的系统,分别对应作业2(信息检索)与作业3(信息抽取)。
两套系统核心全流程**纯 CPU 可运行**(多模态 VLM 增强为可选);**文档语料均为爬虫真实抓取、可逐条溯源,无任何编造数据**。

## 📂 目录结构

| 路径 | 说明 |
|------|------|
| **`作业2-信息检索系统/`** | **作业2 交付件**:BM25 / VSM / 混合检索 + PRF 查询扩展 + 中文 CLIP 跨媒体检索(Qwen-VL 相关性精排) + 人工评价。见其 `README.md` 与 `实验报告_作业2.md`。 |
| **`作业3-信息抽取系统/`** | **作业3 交付件**:集成抽取(正则+触发词+NER 加权投票)7 要素灾害事件 + 知识图谱 + EasyOCR 多媒体抽取 + 逐字段人工评价。见其 `README.md` 与 `实验报告_作业3.md`。 |
| `homework1/` | 作业1(调研报告),与本两系统无关。 |
| `2026信息与知识获取-作业要求与评分标准-*.pdf` | 课程作业要求与评分标准。 |

> 评分以 **`作业2-信息检索系统/`** 与 **`作业3-信息抽取系统/`** 两个自包含文件夹为准,二者完全独立、互不依赖。

## 🚀 快速开始

每个交付文件夹**自包含**(含 `run.py`、`requirements.txt`、`README.md`、实验报告、`src/`、`scripts/`、`data/`),进入任一文件夹即可独立运行:

```bash
# 作业2 — 信息检索系统
cd 作业2-信息检索系统
pip install -r requirements.txt
python run.py web          # http://127.0.0.1:8090

# 作业3 — 灾害信息抽取系统
cd 作业3-信息抽取系统
pip install -r requirements.txt
python run.py web          # http://127.0.0.1:8090
```

> 两套系统默认均使用端口 8090,请**分别单独启动**(勿同时运行)。仓库已附带构建好的索引/向量/抽取结果/OCR 缓存,`python run.py web` 即可直接演示;如需从零重建,见各文件夹 README。

## ✅ 数据真实性

- 作业2:**802 篇**真实新闻(27 域名 / 10 来源);以文搜图图库约 **846 张**(新闻爬取配图 + 325 张主题补充图),经 VLM 质检后 **519 张**入 CLIP 索引。
- 作业3:**118 篇**真实灾害报道(权威源:应急管理部 / 水利部 / 中国地震局 / 四川省政府 / 新华网 / 人民网 / 中新网 / 国家消防救援局 等)、**92 张**灾情配图。
- 每篇文档均含可溯源的原始 `url / source / date`;**已删除一切数据生成器**,无伪造文档。以文搜图图库的主题补充图为真实照片(非编造、非 AI 生成),与文档语料相互独立。
