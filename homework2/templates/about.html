关于信息检索系统

系统架构

本系统是一个完整的中文信息检索系统，包含以下核心模块：

网络爬虫模块
  • 基于 requests + BeautifulSoup 的通用网页爬虫
  • 支持配置种子URL、爬取深度和速率限制
  • 自动提取文章标题、正文、日期、图片
  • 采用礼貌爬取策略，减少对目标服务器的影响
文本预处理模块
  • 基于 jieba 的中文分词
  • 停用词过滤
  • 文本归一化处理
  • 关键词高亮与摘要生成
倒排索引模块
  • 倒排索引数据结构（term → posting list）
  • TF-IDF 权重计算
  • 索引持久化存储与加载
  • 多媒体元数据索引（图片 alt 文本）
检索模块
  • VSM: 向量空间模型 + 余弦相似度
  • BM25: Okapi BM25 概率检索模型
  • 查询扩展（基于共现词分析）
  • 结果排序与摘要生成
创新点
  1. 双算法引擎：同时实现了经典 VSM 模型和改进的 BM25 模型，支持用户在搜索时灵活切换和比较
  2. 查询扩展：基于词共现分析自动扩展用户查询，提升检索召回率
  3. 多媒体信息检索：对文档中的图片元数据进行索引，在搜索结果中展示相关图片
  4. 完善的评价体系：支持 Precision@K、Recall@K、F1、MAP、NDCG 等多种评价指标
  5. 现代化 Web 界面：基于 Flask + Bootstrap 5 的响应式界面，提供良好的用户体验
对环境和社会可持续发展的影响分析
1. 环境影响考量
  • 节能爬取策略：爬虫设置了合理的请求间隔（默认1秒），避免对目标服务器产生不必要的负载，减少网络传输和服务器端能耗
  • 本地化处理：所有数据处理和索引构建在本地完成，无需依赖云端GPU集群，降低了碳排放
  • 轻量级算法：采用经典的 TF-IDF 和 BM25 算法，计算效率高，无需大规模深度学习模型的高能耗训练
  • 索引缓存：构建一次索引可反复使用，避免每次查询都重新处理全部文档
2. 社会影响分析
  • 知识普惠：信息检索系统使人们更高效地获取知识，缩小信息鸿沟，促进教育公平
  • 开源精神：系统基于开源技术栈构建（Python、jieba、Flask），促进技术共享与可持续创新
  • 准确性评价：内置的人工评价机制确保检索质量可量化、可追溯，提升信息获取的可靠性
  • 隐私保护：系统在本地运行，用户的搜索行为和数据不会上传到第三方服务器
3. 可持续发展建议
  • 未来可引入增量索引更新机制，避免全量重建，进一步降低计算资源消耗
  • 可采用更高效的压缩算法存储倒排索引，减少磁盘占用
  • 鼓励使用本系统检索环保、可持续发展相关信息，提升公众环保意识
